# Calculadora de gasto de energia kwh ⚡

 ## Cálculo de gasto de energia mensal de um aparelho 📝

### Linguagem usada: python 🐍 


### Fórmula utilizada: (potencia * horasDia * 30) / 1000 🧮

### Instruções: Digite o nome do aparelho, em seguida a potência e por final o tempo médio de uso diário em horas. O resultado aparecerá em kWh
